# GeekToolStyle.widget

GeekTool-style system information display showing CPU usage, memory, disk, battery, load average, uptime, IP addresses, and top processes.

## Dependencies

Standard macOS CLI tools: `top`, `awk`, `sed`, `bc`, `sysctl`, `df`, `pmset`, `dig`, `ifconfig`, `ps`, `ioreg`, `grep`, `tr`, `uptime`.

These all ship with macOS — no additional installation required.

## Configuration

No configuration needed. The widget works out of the box.
